How to Create a New Template

This page explains the basic ways to create a new item template inside the Item Template Editor, and which approach to choose in different situations.

Before you start

The Item Template Editor always opens with a temporary, unsaved template. This is a blank starting point that allows you to begin designing a new item immediately without creating files manually.

Method 1 — Use the default temporary template

When the editor opens, the first entry you see is a transient (temporary) template. It has no file saved on disk yet and exists only in memory.

To turn it into a real item template:

  1. Fill in the fields on the right panel — name, category, visuals and other data.
  2. Adjust sections like Primary Data, Secondary Data or Technical Data as needed.
  3. Click “Save Template” in the toolbar.

After saving, the item becomes a normal template in your project and appears in the list on the left like any other entry.

Method 2 — Duplicate an existing template

If you want to create something similar to a template you already have, duplicating is usually faster.

  1. Select an existing template from the list.
  2. Use the duplicate option in the toolbar or available controls.
  3. A new copy appears, using the original values as a starting point.
  4. Edit only what needs to change — name, visuals, stats and other properties.

This approach is ideal for creating item families such as multiple potions, weapons with variations, or similar consumables.

Which method should I use?

  1. Use the temporary template when creating a completely new item design from scratch.
  2. Duplicate an existing template when the new item should share most settings with something already created.

After creating a template

Once your new template is saved, you can continue editing it like any other item, export it, or use it inside your inventory systems.
