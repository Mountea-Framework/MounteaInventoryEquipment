How to Export Templates

This page explains the basic ways to create a new item template inside the Item Template Editor, and which approach to choose in different situations.

Saving and exporting

  • After editing a template, click Save Template to save just the active item, or Save All Templates to save all modified entries.
  • You can use Ctrl+S or Cmd+S to save records, too!
  • Use Export Item to create a file that can be shared or imported into another project using Import Template.
