Reporting API Documentation

A compact reference for generating analytics and exportable reports.

Overview

The Reporting API aggregates usage data and delivers summarized results in JSON or CSV.

Base URL: https://api.example.com/reporting

Authentication

Provide an API key via the X-API-Key header.

X-API-Key: YOUR_API_KEY

Endpoints

GET /summary

Returns a time-bounded usage summary.

Query parameters

from
ISO timestamp marking the start of the window.
to
ISO timestamp marking the end of the window.

Example

GET /reporting/summary?from=2025-01-01&to=2025-02-01
X-API-Key: YOUR_API_KEY
POST /exports

Creates a downloadable export.

Request body

{
  "format": "csv",
  "range": { "from": "2025-01-01", "to": "2025-02-01" }
}

Response

{
  "export_id": "exp_987",
  "status": "processing"
}

Status Codes

  • 200 – Success
  • 403 – Invalid or missing key
  • 429 – Rate limit exceeded
